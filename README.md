<h3>Binary Tree</h2>
<ul>
<li>at most 2 children per node</li>
<li>exactly 1 root</li>
<li>exactly 1 path between root and any node</li>
</ul>

<h3>Note</h2>
<ul>
<li>Root node = node has no parent</li>
<li>Leaf node = node has no children</li>
</ul>
